## 描述

PLAN-037 给了人工触发的 RebalancePanel；但 admin 可能数天不打开。本任务加后台 watchdog：周期性运行 `rebalance.Compute()`，若集群持续不均衡（连续 N 次 tick 都 imbalanced），写告警事件 + admin/nodes 顶部 banner 显示，admin 可一键应用建议或 dismiss。

**永不自动迁移**——遵循 PLAN-037 决策，仅"建议 + 人工执行"。

实现：

1. 新表 `system_alerts(id, kind, cluster, severity, payload jsonb, created_at, resolved_at, dismissed_by)`
2. `internal/worker/imbalance_watchdog.go`：5min tick；若 `Compute().Stats.Imbalanced` 持续 ≥ 3 tick（15min）→ 插入 alert（如已有未 resolved 的同 cluster 同 kind 则更新 payload）
3. 新端点 `GET /admin/system-alerts`（active only）+ `POST /admin/system-alerts/{id}/dismiss`
4. admin/nodes 顶部 alert banner：链接到 RebalancePanel 自动展开
