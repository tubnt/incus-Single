## 决策原则

DESIGN.md `@theme` 物理 px 值**不动**（保留 Linear 设计系统完整性）。改的是**组件默认 className 引用的 token 档位**：

| 组件 | 现 | 新 | 理由 |
|---|---|---|---|
| `<PageHeader>` h1 | `text-h1` 32px | `text-h2` 24px | 内页用 marketing 级 32px 偏大；24 与 Stripe/Vercel/GitHub 对齐 |
| `<CardTitle>` | `text-h3` 20px + `font-strong` | `text-body` 16px + `font-emphasis` | 91 处卡片标题统一回调 |
| `<DialogTitle>` | `text-h3` 20 + `font-strong` | `text-body-emphasis` 17 + `font-emphasis` | 对话框略强于卡片 |
| `<SheetTitle>` | `text-h3` 20 + `font-strong` | `text-body-emphasis` 17 + `font-emphasis` | 同 Dialog |
| 数字展示 (余额/月费/金额) | `text-h2` 24 | `text-body-emphasis` 17 | dashboard/launch/billing 三处 |
| `<FormSection>` h2 (launch) | `text-h3` 20 | `text-body` 16 + `font-emphasis` | 段落标题与 CardTitle 一致 |
| 表格列头 | `text-label` 12 | `text-sm` 14 | 提升扫读，对齐 GitHub Primer |
| `<TableCell>` | `py-2` (8) | `py-1.5` (6) + 行高 36 | dense 表格 |
| `<CardHeader>` | `p-5` (20) | `p-4` (16) | dense 卡片 |
| `<PageShell>` | `gap-6` (24) | `gap-4` (16) | 顶级间距 |
| `<PageContent>` | `gap-4` (16) | `gap-3` (12) | 段间 |
| `LaunchPage` 段间 | `gap-8` (32) | `gap-6` (24) | 与社区 24-32 中段对齐 |
| Sidebar 项 | varies | `h-8` (32) + `text-sm` (14) | 与 Linear/Vercel 紧凑 nav 对齐 |
