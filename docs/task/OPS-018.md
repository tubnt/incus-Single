## P2.4 — Floating IP portal attach/detach

### `internal/handler/portal/floating_ip.go`
- `PortalRoutes`：`GET /portal/floating-ips`、`POST /portal/services/{id}/floating-ips/{fipID}/attach`、`POST /portal/services/{id}/floating-ips/{fipID}/detach`
- `PortalListAvailable`：返回所有 status=available 的 FIP + 调用者名下 VM 已绑定的 FIP（用户自己一眼看到状态）
- `PortalAttach` / `PortalDetach`：owner check（`vm.UserID == userID`）→ 调用共享 `attachVerified` / `detachVerified`
- 共享路径加 `via` 字段（`admin` / `portal`）记入 audit log
- `Allocate` / `Release` 仍 admin-only（涉及公网 IP 池配额）

## P2.3 — Admin VM firewall bind/unbind

### `internal/handler/portal/firewall.go`
- 3 个新 admin 路由：`GET /admin/vms/{id}/firewall`、`POST`、`DELETE /admin/vms/{id}/firewall/{groupID}`
- 复用 service 层 `AttachACLToVM` / `DetachACLToVM`（同样的 cold-modify：running VM 自动 stop→PATCH→start）
- 不要 owner check（admin 全权）
- audit `via=admin` 标识
