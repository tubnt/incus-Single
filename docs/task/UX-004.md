# UX-004 新建云主机页面 UX 重做（admin/create-vm + PurchaseSheet）

- **status**: completed
- **completedAt**: 2026-05-02
- **priority**: P2
- **owner**: claude
- **createdAt**: 2026-05-02
- **scope**: 用户决策 "proceed AB"：方案 A（admin/create-vm 结构性重做）+ 方案 B（PurchaseSheet 优化）+ 共享改动（NumberStepper + OsImagePicker → Combobox）

## 背景

调研 DigitalOcean / Linode / Vultr / GCP / Proxmox 创建实例页发现 5 类高优问题：

1. `/admin/create-vm` 五张 Card 套娃（每字段一张），纵向冗余、视觉噪声大
2. 规格预设信息密度过低（`Small/Medium/Large/XLarge` 抽象 label + caption 一行）
3. `OsImagePicker` / `ProjectPicker` 是裸 `<select>`，与 shadcn 风格断层（CLAUDE.md UI 规则要求 headless 库）
4. 批量数量是裸 `<input type=number>`，缺 stepper / 快捷预设 / 总资源汇总
5. `PurchaseSheet` 月价仅出现在 CTA，规格行视觉单薄，未提示 SSH key 自动注入

## 方案

### 共享改动
- 新增 `shared/components/ui/number-stepper.tsx` —— `−/+` 按钮 + 快捷预设 chip。
- 重构 `features/vms/os-image-picker.tsx` —— cmdk + Popover 实现 Combobox（按 Ubuntu/Debian/Rocky/Alma/Fedora/CentOS 等发行版分组、可搜索）。保留 `value/onChange/className/disabled` 接口与 `useOsImageLabel` hook，所有调用点零迁移成本。

### A. admin/create-vm 页结构性重做（`web/src/app/routes/admin/create-vm.tsx`）
- 去除每字段 Card 外壳，改 single panel + section 标题分段（Compute / Image / Place / Identify）。
- 规格预设卡片重做：大字号 vCPU/RAM/SSD + 用例 hint；保留 OPS-013 active 视觉与 `data-testid` 不动。
- 批量数量改 `NumberStepper` + 快捷 1/4/8/16 + 实时总资源汇总。
- 新增可选 hostname 前缀字段（仅前端拼接预览，后端协议不动）。
- Summary 卡：count > 1 时聚合 "总计 N×4 vCPU / N×4 GB / N×50 GB"；移除 desktop 内嵌主按钮（避免双 CTA），desktop 主按钮落到表单底部。
- 创建结果改为替换主区域（不再叠加），凭据网格化 2 列。

### B. PurchaseSheet 优化（`web/src/features/billing/purchase-sheet.tsx`）
- Sheet 顶部突出月价 + 套餐名（产品卡同款大字）。
- Spec 行用图标化 3 格替代当前文字网格。
- OS 镜像同步换成新 Combobox。
- 新增不显眼提示："已绑定 N 把 SSH Key 将自动注入" 链 `/ssh-keys`。

## DESIGN.md 强约束
- 禁 hex 字面量与 arbitrary value，所有视觉值走 `index.css @theme` 暴露的 CSS 变量。
- DESIGN.md 优先级高于 pma-web。
- 颜色：`bg-surface-1/2/3`、`text-text-primary/secondary/tertiary`、`border-border`、`bg-primary` etc。
- 半径：`rounded-md/lg/xl`（依 §5.4 量表）。
- 字号：`text-h3 / text-body / text-caption / text-label`，默认字重 `font-emphasis`(510)。
