## 修复模板

与 Bug #7 / #13 同样模式：

```tsx
<button
  aria-label={`Delete <type> ${name}`}            // 选 selector 稳定，不走 i18n
  data-testid={`delete-<type>-${slug}`}
  className="... border border-destructive bg-destructive/20 text-destructive"
  onClick={...}
>
  ⚠ Delete
</button>
```

特殊情况：
- **Shadow login** (`/admin/users`)：aria-label 模板 `Shadow login as ${username}`，testid `shadow-login-${userId}`；建议 confirm dialog 文案显式 `Login as <user>?` 而非 `Shadow?`
- **Evacuate** (`/admin/ha`)：aria-label `Evacuate node ${nodeName}`，testid `evacuate-node-${nodeName}`；confirm 必须列出该节点上 VM 数量与名字
