## Summary

用户在生产环境反馈「用户版面我做一下全业务流程测试,发现不少问题,例如开云主机开不了、余额支付提示 404 等等」。

本次对 `vmc.5ok.co` 用户端进行一轮全量 E2E:Dashboard → VM 列表/详情/控制台/快照/重置密码 → 账单/订单/发票 → SSH 密钥 → API Token → 工单 → 设置 → i18n 切换 → 主题切换 → 404。

## Findings (N1-N18)

### P1 — 影响用户认知或产生 403/404 噪声

- **N1** `/vm-detail` 监控面板标签硬编码中文(内存/磁盘/网络)——英文用户看到混排
- **N2** `/console` 页「← Back to VMs」硬编码跳 `/admin/vms`——普通用户触达 403(PLAN-015 J-P2.1 已修大部分 portal/admin 前缀,console 页遗漏)
- **N3** VM Stop / Restart 操作返回 200 后,portal 的 VM 详情页状态标签不刷新(显示仍「running」),但 admin 侧同 VM 显示「Stopped」——React Query cache 未按路径维度 invalidate
- **N4** Snapshot Delete 成功后,列表不自动刷新,需要切换 Tab 或 reload

### P2 — i18n 覆盖不全(PLAN-015 N3 巡检遗漏)

- **N5** `/ssh-keys` 页「添加密钥」按钮硬编码中文
- **N6** `/ssh-keys` 表头「添加于」硬编码中文
- **N7** `/api-tokens`「Create API Token」按钮显示为「+ + Create API Token」(双 +)
- **N8** `/api-tokens` 表头「创建」「最后使用」硬编码中文
- **N9** API Token 创建成功 toast 硬编码中文
- **N10** top-bar 图标按钮 aria-label「Language」「Logout」硬编码英文,zh 下未切换

### P3 — UX 润色

- **N11** Reset Password toast 仅显示密码,未提供复制按钮

### 功能缺口(非 bug,列入 Backlog)

- **G1** `/billing` 页无余额卡 + 无充值入口(Dashboard 有 $7.00 但 billing 页无)
- **G2** `/billing` 订单 Actions 列为空(无取消/退款)
- **G3** `/billing` 发票无详情/下载动作
- **G4** `/tickets` 用户无 Close Ticket 按钮
- **G5** Dashboard 无快捷操作入口(Create VM / Top-up)
