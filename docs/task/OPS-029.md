## Description

OPS-028 之后做全功能 UI 测试发现 EN locale 下大量字符串仍是中文：`en/common.json` 多个 sub-object（admin.firewall / admin.floatingIPs / admin.osTemplates / admin.users / common / shadow / billing / sshKey / ticket / topbar / vm 等）的 value 是中文 placeholder。同时 DataTable 用了未在 zh 定义的 key，audit-logs 的 User 列还在显示数字 ID（同 OPS-028 P3.2/3.3 同根因），sidebar 折叠按钮 aria-label 硬编码中文。

## Acceptance criteria

- en/common.json 不再有非符号类的中文 value（剩 ✓ ⏸ ▶ ← 等纯符号 OK）
- DataTable 的 `selectAll` / `selectRow` / `resizeColumn` key 在 zh + en 都定义
- admin/audit-logs 的 User 列展示 email（不是数字 ID）
- sidebar 折叠按钮 aria-label 走 i18n
- 后端 unit + 前端 tsc + build 全绿
